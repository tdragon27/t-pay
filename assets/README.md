# Assets

## Fonts
Place the following font files in this directory:
- `SpaceMono-Regular.ttf`  — available from Google Fonts (https://fonts.google.com/specimen/Space+Mono)
- `SpaceGrotesk-Regular.ttf` — available from Google Fonts (https://fonts.google.com/specimen/Space+Grotesk)

## Images
The following placeholder images are required by `app.json`:
- `icon.png`         — 1024×1024 app icon
- `splash.png`       — 1284×2778 splash screen (or any size)
- `adaptive-icon.png`— 1024×1024 Android adaptive icon foreground
- `favicon.png`      — 48×48 web favicon

You can generate these quickly with any image editor or use Expo's default assets
temporarily by copying them from `node_modules/expo/assets/`.
