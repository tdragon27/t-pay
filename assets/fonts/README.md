# T Pay local fonts

Fonts are bundled with the app; no runtime CDN request is made.

- `GeneralSans-Semibold.ttf` and `GeneralSans-Bold.ttf`: Fontshare CDN,
  General Sans by Indian Type Foundry. License:
  https://www.fontshare.com/licenses/itf-ffl
- Inter: Google Fonts distribution, SIL Open Font License 1.1.
- Space Mono: Google Fonts distribution, SIL Open Font License 1.1.

Source pages:

- https://www.fontshare.com/fonts/general-sans
- https://github.com/rsms/inter
- https://github.com/googlefonts/spacemono
